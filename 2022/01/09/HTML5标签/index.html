基础标签

标签 描述
<!DOCTYPE> 定义文档类型
<html> 定义 HTML 文档
<head> 定义关于文档的信息
<title> 定义文档的标题
<body> 定义文档的主体
<h1> to <h6> 定义 HTML 标题
<p> 定义段落
<br> 定义简单的折行
<hr> 定义水平线
<!--...--> 定义注释

<!DOCTYPE>

必须是 HTML 文档的第一行，位于 标签之前

指示 web 浏览器关于页面使用哪个 HTML 版本进行编写的指令
